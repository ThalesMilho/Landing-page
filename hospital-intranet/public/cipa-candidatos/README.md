# Instruções para substituir as imagens dos candidatos CIPA

Este diretório contém as imagens dos candidatos da CIPA. Para substituir os placeholders pelas imagens reais:

## Arquivos existentes:

2. **andresa.png** - Andressa Macedo do Carmo Nascimento (Analista de Prestação de Contas)

6. **vanessa.png** - Vanessa José da Silva (Coordenadora de Atendimento)

## Como substituir:

1. Localize a imagem correspondente a cada candidato
2. Substitua o arquivo existente pela imagem real
3. Mantenha o mesmo nome do arquivo
4. Use formato PNG para melhor compatibilidade

As imagens serão exibidas automaticamente na página inicial da intranet.
